### 14. 문자열 이어 붙이기



#### 문자열 길이에 접근하기

```js
var str = "Hello";
str.length; // 5
str["length"]; // 5
"Hello".length; // 5
```



#### 두 문자열 이어붙이기

```js
var str = "World"; // undefined
var str2 = str.concat(str2); // "HelloWorld"
str2; // "HelloWorld"
```



#### 문자열 여러 개를 이어 붙이고 싶다면?

```js
str.concat(str2).concat("!"); // "HelloWorld!"
"Hello".concat("World").concat("!"); // "HelloWorld!"
```

```js
str+str2; // "HelloWorld"
"Hello"+"World"; // "HelloWorld"
"Pi is " + 3.14; // "Pi is 3.14"
```
